Qu'est ce que PhpLdapAdmin :

PhpLdapAdmin adapté au LCS, permet à l'administrateur de gérer l'annuaire LDAP via une interface web.

  • Arborescence LDAP
  • Copier les entrées LDAP (même exemplaire entre les différents serveurs)
  • Copier récursivement des arbres entiers
  • Supprimer les entrées LDAP
  • Supprimer récursivement
  • Afficher et modifier les attributs de l'image (comme jpegPhoto)
  • Recherches LDAP ( simple et avancée)
  • Exportation LDIF et DSML
  • Importation LDIF
  • Renommer les entrées LDAP
  • Gérer les hachages de mots de passe utilisateur (supports sha, crypt, md5, blowfish, md5crypt
  • Augmentation automatique des numéros UID
  • Configurable en lecture seule et lecture / écriture modes.
