Documentation LCS squid

Version : Paquet lcs-squid 2.4.8

Configuration du service de cache mandataire basé sur Squid pour LCS.

Reconfiguration de squid

Suite à l'installation du paquet lcs-squid, il est possible de reconfigurer Squid grace au script /usr/share/lcs/scripts/squid-reconfigure.

Usage :

squid_reconfigure.sh 
    <DOMAIN> <CACHEDIRSIZE> <VLANSC> <VLANADMINISTRATIF> <VLANPEDA1> \
    <VLANPEDA2> <VLANPEDA3> <VLANPEDA4>

D&eacut;finitions des param&egrav;tres :

  • <DOMAIN> : Nom de domaine,
  • <CACHEDIRSIZE> : Taille du cache en MB,
  • <VLANSC> : Adresse/masque du Vlan service commun,
  • <VLANADMINISTRATIF> : Adresse/masque du VLAN ADMINISTRATIF,
  • <VLANPEDA1> : Adresse/masque du VLAN pédagogique n°1,
  • <VLANPEDA2> : Adresse/masque du VLAN pédagogique n°2,
  • <VLANPEDA3> : Adresse/masque du VLAN pédagogique n°3,
  • <VLANPEDA4> : Adresse/masque du VLAN pédagogique n°4.
