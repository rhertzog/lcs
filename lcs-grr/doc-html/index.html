Plugin LCS Gestion et réservation de ressources

Fonctionalités du système de Gestion et réservation de ressources :

Intégré au serveur de communication LCS, le module de Gestion et réservation de ressources est basé sur l'application Grr.

Remarques:

Il est INDISPENSABLE que l'administrateur accède à Grr juste après la mise à jour, pour lancer le script de mise à jour de la base de données

Dernière version experimentale :

Version module : 2.4.10~4 du 07/05/2015 basée sur la version 1.9.7e de Grr
