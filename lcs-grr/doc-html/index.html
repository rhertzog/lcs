Plugin LCS Gestion et réservation de ressources

Fonctionalités du système de Gestion et réservation de ressources :

Intégré au serveur de communication LCS, le plugin de «Gestion et réservation de ressources» est basé sur l'application Grr.

Remarques:

En cas de mise à jour depuis la version 0.3 du plugin, il est INDISPENSABLE que l'administrateur accède à Grr juste après la mise à jour, pour lancer le script de mise à jour de la base de données

Dernière version stable :

Version plugin : 0.4 du 02/12/2008 basée sur la version 1.9.5c de Grr
