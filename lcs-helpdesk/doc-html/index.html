Documentation Module LCS HELPDESK

Qu'est ce que lcs-helpdesk :

lcs-helpdesk , permet aux administrateurs via une interface web de :

  • Une authentification transpatrente en SSO sur la plateforme de HelDesk academique, ...
  • Un accès natif aux fonctionnalités offertes par l'API du HelpDesk pour, ...
  • S'enregistrer a partir du LCS, ...
  • Deposer un nouveau ticket de maintenance, ...
  • Consulter l'evolution de la gestion des tickets, ...
  • Repondre a un message depose par l'equipe TICE academique, ...
