Documentation LCS Pré-Installation

Version : Paquet lcs-preinstall 2.4

Pré-Installation LCS sur Debian Etch.

Fichier requis : /etc/lcs/lcs.conf

Pour une installation «silencieuse», il faut mettre en place le fichier /etc/lcs/lcs.conf préalablement à l'installation de ce paquet.

Un fichier lcs.conf exemple, est disponible sur le svn LCS.

Actions :

  • Installation de : slis-common, sudo, make, pwgen, ntpdate, mysql-server, at
  • Configuration des locales en FR_fr
  • Mise en place des paramètres LCS dans la base lcs_db.params
