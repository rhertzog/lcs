Apache2 SSL LCS, renouvellement du certificat serveur

A l'installation du module lcs-apache2ssl, un certificat est généré pour une péridoe de un an.

Il sera nécessaire de regénérer le certificat une fois cette période arrivée à son terme.

Cette action de régénération du certificat est possible, via l'interface web d'administration et le menu : Administration >Services Intranet >Certificat Apache SSL ou en ligne de commande grace au script /usr/share/lcs/scripts/apache2-ssl-cert.sh.

Usage :

apache2-ssl-cert.sh
    <FQN> <DOMAIN> <COUNTRYNAME> <PROVINCENAME> 
    <LOCALITYNAME> <ORGANIZATIONNAME> <ORGANIZATIONALUNITNAME>

Définitions des paramètres :

  • <FQN> : Nom de domaine pleinement qualifié
  • <DOMAIN> : Domaine
  • <COUNTRYNAME> : Nom du pays sur deux lettres (ex: FR)
  • <PROVINCENAME> : Nom de la province (ex: Basse Normandie)
  • <LOCALITYNAME> : Nom de la ville
  • <ORGANIZATIONNAME> : Nom de l'établissement
  • <ORGANIZATIONALUNITNAME> : Nom de l'unité organisationnelle (ex : Lycée, collège...
