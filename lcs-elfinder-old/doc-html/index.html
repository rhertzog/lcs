Limitation des access :

Pour limiter l'accès uniquement depuis l'intranet de l'établissement au répertoire public_html ou à l'un de ses sous-répertoires, il est possible de créer une fichier .htaccess.

Synthaxe du fichier .htaccess :

Le fichier .htaccess à déposé dans le répertoire à protéger sera :

#lcs-restrictions d'acces
order deny,allow
deny from all
allow from @IPLAN@
errordocument 403 "../Err/403.html"

Comment créer le fichier .htaccess à avec elFinder :

  • Créer un fichier de nom .htaccess.txt
  • Editer le fichier et copier / coller le contenu ci-dessus
  • Renommer le fichier .htaccess.txt en .htaccess
