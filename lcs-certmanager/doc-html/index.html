Documentation LCS CERT MANAGER

Version : Paquet lcs-certmanager 2.4.9

Gestion des certificats SSL LCS

Les services apache2 SSL, imap SSL, CAS, ont besoin d'un certificat SSL, le paquet lcs-certmanager, permet de mettre en place un certificat SSL ou à défaut, de générer un ceetificat SSL auto-signé.

Fonctionnalités de LCS Certificats Manager

  • Génération d'un certificat SSL auto signé à l'installation du paquet
  • Possibilité de regénéré le certificat auto signé pour une période de 365 jours
  • Import d'un certificat SSL (à faire)
  • Supprimer un certificat SSL (à faire)
  • Consulter la liste des certificats disponibles (à faire)
  • Sélectionner un certificat SSL (à faire)
