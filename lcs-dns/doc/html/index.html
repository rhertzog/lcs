Installation du service DNS sur LCS.

Reconfiguration du service DNS

Suite à l'installation du paquet lcs-dns, il est possible de reconfigurer le service DNS grace au script /usr/sbin/dns-reconfigure.sh.

Usage :

dns_reconfigure.sh 
    <DNS_PRIMARY> <DNS_SECONDARY> <IPADDR> <HOSTNAME> <DOMAIN>

Définitions des paramètres :

  • <DNS_PRIMARY> : Serveur DNS primaire
  • <DNS_SECONDARY> : Serveur DNS secondaire
  • <IPADDR> : Adresse IP LCS
  • <HOSTNAME> : Nom de d'hote LCS
  • <DOMAIN> : Nom du domaine LCS
