Documentation LCS monlcs

Version : Paquet lcs-monlcs 2.1-1

Mon LCS est un portail pédagogique permettant une appropriation individuelle de ressources et un partage de ressources au sein d'un établissement scolaire.

Mon LCS est basé sur la technologie Ajax versus ( prototype.js ) et effets DHTML adaptés pour créer un environnement le plus intuitif que possible (Environnement de bureau avec fenêtres de ressources, gestions des ressources, notes personnelles, flux RSS )

PRESENTATION

MonLCS pourrait être défini comme un portail Ajax parcequ'il intègre le mécanisme des requêtes asynchrones.
Concrètement l'utilisateur n'a pas à recharger l'ensemble de la page pour rafraichir le contenu de la page. Seuls quelques fragments HTML de la page sont
rechargés pour interagir avec l'utilisateur.

Cela permet d'offrir un contexte de travail qui se rapproche de ce qu'un utilisateur peut trouver sur tout poste informatique. Les applications Web peuvent
désormais grâce à la technologie Ajax  tendre de plus en plus vers une utilisation de l'application qui se rapproche de plus en plus des applications classiques
sur un poste de travail.

Mon LCS intègre une philosophie Web 2.0 au sens ou l'utilisateur se retrouve au coeur du système et ne se limite pas à un rôle de consommateur d'information
mais devient à son tour contributeur. Concrètement il peut personnaliser, proposer, enrichir et interagir  avec le contenu proposé.

Dans un contexte scolaire, mettre des ressources à disposition des professeurs, des élèves, des parents ou tout autre membre de la communauté éducative
est une évidence. L'existence de l'annuaire LDAP sur les LCS permet  de regrouper  ces individus selon une organisation propre à chaque établissement.
L'intégration de Mon LCS au LCS est donc assez naturelle. L'installation est très simple, il suffit d'installer le paquet module lcs-monlcs qui fournit par défaut
un pot de ressources de départ. Aucun paramètrage n'est nécessaire. Dès l'installation tous les utilisateurs de l'annuaire LDAP ont par défaut un accès à
Mon LCS avec un profil propre.

Les élèves ont essentiellement un profil de consultation et personnalisation.
Les professeurs peuvent au sein des equipes pédagogiques et des classes proposer des ressources.
Le personnel administratif peut aussi partager des ressources à destination des élèves ou des professeurs.
Le gestionnaire qui détient le droit monlcs_is_admin a quant à lui pour ainsi dire pouvoir absolu.

L'entité de base de Mon LCS est une ressource. Qu'est ce qu'une ressource ? Techniquement nous pourrions répondre qu'une ressource est un média lisible
dans un navigateur (Document HTML, Animation flash, Applet java, Effets DHTML/javascript, son, video), sous réserve que les divers plugins soient
disponibles pour les types d'informations citées ci-dessus. Première conséquence il faudra activer le javascript sur le navigateur cible.
La présence du plugin Flash s'avère aussi être indispensable. Le navigateur Mozilla Firefox (ou tout navigateur compatible) est fortement recommandé.
Outre son excellent respect des standards CSS et javascript, il est fréquemment mis à jour et dispose d'une bonne sécurité. Une utilisation d'Internet Explorer
reste toutefois possible notamment pour les élèves et les parents.

ORGANISATION

Pour gèrer au mieux chaque type de ressource une organisation par onglets à été retenue.
Chaque onglet possède une série de menus associés.

Le gestionnaire de MonLcs (qui détient le droit monlcs_is_admin) peut ajouter des onglets "établissement".

Pour tous ces onglets le menu  Proposer: permet à un utilisateur avec droits de proposer une ressource dans le contexte donné.

Tout utilisateur hormis les élèves, peut ajouter un onglet personnalisé. Il n'existe pour l'instant aucun mécanisme de partage de menus.

Le menu Accueil :



    Mon bureau LCS: espace personnalisé c'est un bureau de travail et/ou un bac à sable.
    Actualités: contient pour l'instant l'ancien écran MonLcs
   

Le menu Ressources :



    Ressources pédagogiques: contient une liste de ressources classées par matière.
    Fils RSS : permet aux utilisateur de choisir/personnaliser une série de flux d'information (fils RSS) pour être tenu informé d'une actualité
    Ressources académiques: permettra de proposer des ressources propre à l'académie
    Vie scolaire: permettra le partage d'information à caractère plus administratif

Le menu Applications :



    Outils LCS: permet de selectionner les plugins/modules LCS favoris.
    Couteau suisse: ensemble utile et/ou futile de gadgets (Horloge/Montre/calculatrice/piano)

Le menu Scénarii pédagogiques :



    Choisir: liste des scenarios classés par matières ou modification incrémentielle d'un scénario existant et espace de construction d'un nouveau scénario.

Le menu Aide :


  
    Guide d'utilisation: aide au format livre numérique
    Proposer: permettra de proposer des aides/tutoriels à une équipe, une classe ...

Ajout de menu  :



     (En Rouge) Possibilité pour  le gestionnaire de MonLCS d'ajouter des onglets établissement (CDI , Divers pour les jeux etc)
     (En vert) Possibillté pour un utilisateur n'aynat pas le statut d'élève d'ajouter des onglets personnels.

La gestion des ressources

Pot de ressource: est un pot qui regroupe les ressources publiques (accessibles pour tous) et privées.

       

Notez le message qui indique si vous disposez du droit monlcs_is_admin.
Affiche en popup une courte description. Un double-clic permet à l'auteur de modifier la description.
Efface la ressource en profondeur dans toutes les tables pour monlcs_is_admin.
Un double clic change le statut
private ( privée )<-> public ( publique )
Comment la ressource est-elle utilisée ?
Les jumelles permettent de placer une fenêtre pour voir la ressource.

Le crayon permet de renommer la ressource/

Le menu droit contextuel

Chaque menu hérite d'un menu contextuel adapté dynamiquement en fonction du profil utilisateur.
Ce menu contextuel est accesible par clic droit sur une zone libre de l'écran.
Le bouton clic droit virtuel le fait apparaître également.

Enregistrer le bureau (TOUS) Cette commande sauvegarde l'état du bureau dans les bases de données de  MonLCS.
Proposer par defaut (monlcs_is_admin) Fixe les ressources qui apparaissent lorsque aucune ressource n'est enregistrée dans le menu en cours.
Ressources (TOUS) Fait apparaître le pot de ressources.
Ajouter une note (Dans Bureau ou Scenari pour les professeurs et dans les onglets personnels)
 Permet l'ajout d'un note avec l'editeur FCKEditor configuré pour interagir avec le home de l'utilisateur. Cet editeur HTML permet d'incorporer des images, de créer des liens etc.
Figer des ressources (monlcs_is_admin) Cette fonction force les ressources sur un menu. Elle est en contradiction avec la fonction proposer par défaut. Il est fortement  conseillé de ne pas en abuser !
Publier un scénario dans l'onglet Scenarii Sauvegarde l'ecran en cours sous la forme d'un scénario pédagogique.


Ajouter une ressource ( indispensable AVANT de proposer )



Règle de base : TOUTE RESSOURCE DOIT ETRE AJOUTEE AVANT D'ETRE PROPOSEE.
Si la ressource est publique tous y ont accès. Pour restreindre l'accès donnez un statut privé à la ressource
puis proposez là dans un menu donné.

Ce menu n'apparaît que si vous en avez le droit.
Donner l'URL exemple http://trucmuche.com
Donner un titre.
Donner une courte description.
La ressource est publique ? Cochez la case Rendre la ressource publique.
La ressource est un fil RSS ? Cochez la case La ressource est un flux RSS.

Miniature - vignette



Il est posssible, un peu comme sur Exalead, de choisir une vignette (sorte de raccourci) pour la ressource qui apparaîtra en lieu et place de la ressource.
Pour le choix de la miniature:  trois options

    aucune miniature (animation - widget)
    miniature générée par un service web externe
    upload d'une vignette "maison"

le boution vérifier permet de se rendre compte de la validité de la vignette.

La taille de la miniature est conditionnée par la configuration php.

IL EST CONSEILLE D'ALLEGER AU MAXIMUM LA TAILLE DE CETTE IMAGE.

(diminuer la taille 200px sur 150px par exemple et diminuer le nombre de couleurs).
L'image ne devrait pas dépasser une centaine de Ko.

Prévisualisez avec l'oeil.

UTILSER Mon LCS.        

La présentation donnée ci-dessus se voulait relativement succinte. Elle visait à fournir les informations de première minute néessaires à une prise en main rapide de l'interface MonLCS.
Pour approfondir un aspect  précis, reportez vous ci-dessous en fonction de votre profil.

    1° Le profil élève. (et parent indiretement)

        a) Personnaliser son bureau.
        b) Les ressources publiques.
        c) Les ressources proposées.
        d) Les fils RSS.
        e) Les scénarios pédagogiques.

    2° Le profil professeur et administratif.

        a) Le menu personnel.
        b) Gestion des notes personelles.
        c) Proposer une ressource.
        d) Gestion des scénarios pédagogiques. (créér - mettre à jour - supprimer )
        
    3° Le profil gestionnaire monlcs_is_admin.
        a) Quel est son rôle ?
        b) La gestion des onglets établissement. (création - adaptation - suppression)
        c) La suppression d'une ressource équivoque.
