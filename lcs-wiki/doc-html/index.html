Plugin LCS WIKI

Fonctionnalités du plugin LCS «WIKI» :

Intégré au serveur de communication LCS, le plugin «WIKI» est basé sur l'application WikiniMST.

Il permet :

  • l'écriture collaborative de documents

  • la création d'une base de connaissances

  • la gestion de projets au sein de la classe

  • la gestion de projets inter-classe / intergroupe

  • la gestion de projets pédagogiques et/ou administratifs

Remarques:

  • L'administrateur peut créer les utilisateurs et les groupes à partir de "Gestion des comptes" du menu gauche. Les comptes sont créés automatiquement à partir de l'annuaire LDAP.

  • Il peut également activer/désactiver les modules complémentaires à partir de "Administration générale " du menu gauche.
