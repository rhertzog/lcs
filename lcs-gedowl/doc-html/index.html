Plugin LCS de Gestion Electronique de Documents

Fonctionnalités générales du plugin LCS «Gedowl» :

Ce plugin de Gestion Electronique de Documents (GED) basé sur OWL (http://owl.sourceforge.net/.)
a été adapté et pré-configuré pour une utilisation sur LCS dans un établissement scolaire.

Owl est un système de gestion de documents multi-utilisateurs. Les utilisateurs ont la possibilité de déposer des documents et de leur assigner des attributs(permissions, commentaires, etc ..). Les autres utilisateurs sont alors en mesure de localiser les documents, soit en utilisant l'arborescence des dossiers ou en utilisant le moteur de recherche interne.
Il est donc particulièrement adapté pour le travail collaboratif

Remarques:

Pour que le plugin «Gedowl» soit utilisable, l'administrateur doit y accéder préalablement pour importer les utilisateurs.

ATTENTION : Comme il n'est pas possible de faire suivre les Acls (Droits) sur les fichiers et repertoire d'une année à l'autre, le dispositif est entièrement réinitialisé en début d'année scolaire lors de l'import des utilisateurs.
