Configuration du service de courrier smtp basé sur Postfix pour LCS.

Reconfiguration de Postfix

Suite à l'installation du paquet lcs-smtp, il est possible de reconfigurer Postfix grace au script /usr/share/lcs/scripts/postfix-reconfigure.

Usage :

postfix_reconfigure.sh 
    <FQDN> <DOMAIN> <LOCAL_DOMAIN> <NETWORK> <LDAP_SERVER> <BASEDN> <RELAYHOST> <XEN_NETWORK>

Définitions des paramètres :

  • <FQDN> : Nom de domaine pleinement qualifié
  • <DOMAIN> : Domaine LCS
  • <LOCAL_DOMAIN> : Domaine local de courrier
  • <NETWORK> : Complete la variable mynetworks de Postfix (Postfix relaie le courrier des clients des réseaux autorisés et des domaines autorisés. Les réseaux autorisés sont définis par le paramètre mynetworks.)
  • <LDAP_SERVER> : Adresse IP du serveur LDAP
  • <BASEDN> : Base DN du serveur ldap pour renseignement de ldap-aliases.cf
  • <RELAYHOST> : Passerelle de messagerie
  • <XEN_NETWORK>: Réseau XEN
