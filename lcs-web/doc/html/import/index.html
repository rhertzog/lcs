Importation des données de l'annuaire

Principe

Le fonctionnement du Lcs et de SambaÉdu est fondé sur l'utilisation d'un annuaire des utilisateurs et des différents groupes de travail associés.

Il est possible d'importer les informations nécessaires à la constitution automatique de l'annuaire Ldap des acteurs de l'établissement scolaire, donc du système :

  • depuis les fichiers CSV/XML de Sconet et Sts web,
  • ou bien depuis un ensemble de fichiers texte correctement formatés.

    • Ces fichiers texte peuvent être également générés depuis des exports Sconet / Sts web. Voir cette page

Importation

  • Via des fichiers CSV/XML de Sconet et Sts web
  • Via des fichiers texte
