4. Mode administrateur

Le mode administrateur pour permet de configurer l'application, cette fonctionalité est disponible dans votre menu (configuration) si le droit system_is_admin vous a été affecté par l'administrateur du serveur LCS.

Figure 7. Menu Configuration

4.1. Mail

Vous permet de renseigner soit :

  1. Une adresse électronique d'un utilisateur,

  2. Une liste d'adresses électroniques d'utilisateurs séparées par des virgules,

  3. Une adresse électronique correspondant à un groupe de l'annuaire LDAP (par exemple support_info).

Les demandes de support et réponses aux demandes seront diffusées à ces adresses, il convient donc de choisir les adresses des membres du droit Maint_is_admin ou de choisr un groupe LDAP (par exemple support_info) renseigné avec les membres du droit Maint_is_admin dans ce dernier cas il faudra que la configuration du serveur de courrier Postfix supporte les listes de diffusion LDAP.

4.2. Secteur

Vous permet d'ajouter, supprimer, moifier les secteurs de l'établissment.

Figure 8. Menu Secteur

4.3. Topologie

Vous permet d'importer la structure de l'établissement (batiments, étages, salles).

Figure 9. Import et consultation de la Topologie de l'établissement

Le format d'importation doit avoir la structure suivante :

Nom_de_l_etablissement;Bâtiment repere_du_Batiment;Etage;Numero_salle (description) ;0
...
Pierre et Marie Curie;Bâtiment E;1er;E101 (salle de cours);0
...

Elle est conforme au fichier config.txt du logiciel d'inventaire du CR de basse normandie.

4.4. Sauvegarde / Restauration de la base de données

Vous permet de sauvegarder ou de restaurer un fichier sql de la base de données de l'application.
