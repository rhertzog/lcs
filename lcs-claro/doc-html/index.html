Module LCS CAROLINE

Fonctionalités du système de Plateforme de formation :

Intégré au serveur de communication LCS, le module de Plateforme de formation est basé sur l'application Claroline qui est une plate-forme Open Source de formation à distance et de travail collaboratif. Elle permet aux formateurs de créer des espaces de cours en ligne et de gérer des activités de formation sur Internet

Version :

Version module : 2.4.8~4 basée sur la version 1.10.6 de Claroline.

Avertissements :

  • Lors de l'installation du module, si le plugin LCS est détecté, un clône de cette application (fichiers et base de données) est créé, puis mis à jour d'abord vers la version 1.9, puis vers la 1.10 dans un second temps.
    Ces opérations peuvent Ãªtre très longues si la plateforme contient un nombre important de Cours.
    Soyez donc très patient ! (compter environ 1 minute par cours )
  • A l'issue de la mise à jour, le plugin est désactivé, mais est resté intact. Donc en cas de problème avec le module, il suffit de désinstaller le module et réactiver le plugin dans la page d'administration des plugins installés.
  • Le module utilise le dispositif d'authentification CAS. Il faut IMPERATIVEMENT que le port 8443 du routeur soit ouvert.
  • Si le module lcs-desktop est installé, les icônes des utilisateurs sont mis à jour pour pointer vers le module
