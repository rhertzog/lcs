Squelette LCS :

Le squelette LCS de SPIP, est désormais installé dans un répertoire dédié situé en : /usr/share/lcs/spip/lcs_skel/ L'utilisateur spip.manager possède les droits d'écriture sur ce répertoire et peut par conséquent adapter le squelette à ses besoins.

Filtres Audio et Vidéo dans les articles (une autre méthode est disponible dans le nouveau squelette)

Deux expressions peuvent etre intégrées dans un article :

  • flashflv : Pour diffuser un document au format flv dans un lecteur flash,
  • flashmp3 : Pour diffuser un document audio au format mp3 dans un lecteur flash.
Balise audio :
<flashmp3=login_utilisateur/nom_du_document.mp3>Description</flashmp3>

Le document au format mp3 devra se trouver dans le répertoire public_html de l'utilisateur "login_utilisateur".

Balise vidéo :
<flashflv=login_utilisateur/nom_du_document.flv>Description</flashflv>

Il est a noter que le document sera diffusé dans une taille de 200x160 pixels et que si il existe dans le même répertoire (le public_html de l'utilisateur) un fichier image du même nom que le document flv, cette image apparaitra dans le lecteur tant que la lecture ne sera pas demandée.

Comment obtenir un document au format flv ?

A partir d'un fichier avi, mpg...

Convertir celui au format flv avec ffmpeg

Commande : ffmpeg -i video.avi -ab 56 -ar 22050 test.flv

Utilisateur spip.manager

Un utilisateur (login: spip.manager) est disponible, son répertoire home est localisé en /usr/share/spip/ et il possède les droits d'écriture sur le répertoire du squelette lcs-skel. Cet utilisateur est pourvu du shell sftp-server et son mot de passe a été initialisé avec une valeur aléatoire lors de l'installation du paquet Debian lcs-spip. Pour pouvoir utiliser ce compte, il faudra réinitialiser son mot de passe à l'aide de l'interface web d'administration LCS.

Sauvegarde et restauration de la base de données SPIP

En se connectant en sftp à l'aide du compte spip.manager, il sera possible de créer le fichier adminxxxx demandé pour la confirmation de la sauvegarde de la base de données SPIP.
