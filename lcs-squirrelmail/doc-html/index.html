Reconfiguration de squirrelmail

Suite à l'installation du paquet lcs-squirrelmail, il est possible de reconfigurer squirrelmail grace au script /usr/share/lcs/scripts/squirrelmail-reconfigure.sh.

Usage :

squirrelmail_reconfigure.sh 
    <LDAP_SERVER> <LDAP_BASE_DN>

Définitions des paramètres :

  • <LDAP_SERVER> : Adresse IP du serveur LDAP
  • <LDAP_BASE_DN> : Base DN de l'annuaire
