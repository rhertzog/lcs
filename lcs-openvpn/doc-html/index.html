Documentation OpenVPN

Version : Paquet lcs-openvpn 2.4.2

Configuration du service VPN OpenVPN base sur OpenVPN pour LCS.

Reconfiguration de courier-imap

Suite a l'installation du paquet lcs-openvpn, il est possible de reconfigurer openvpn grace au script /usr/share/lcs/sbin/lcs-openvpn-reconfig.

Usage :

lcs-openvpn-reconfig  <port> <vpn-network> 

 <vpn-network-mask> <dh>  <vlan>  <nb-vpn-connexions>  <client-to-client-> 

  <cert>  <masquerade> <wins> <dnsclient>

On retrouve ces meme parametre sur l'interface web

Definitions des parametres :

  • <port> : Port UDP des connexions vpn
  • <vpn-network> : Adresse reseau utilisee une fois la connexion etabli, doit etre un nouveau reseau
  • <vpn-network-mask> : Masque du reseau VPN
  • <dh> : Valeur de hashage pour la generation des certificats (eviter de changer cette valeur)
  • <vlan> : prend pour valeur 0 ou 1, 1 pour activer la propagation des vlan sur le VPN
  • <nb-vpn-connexions> : nombre de clients simultane possible
  • <client-to-client> : prend pour valeur 0 ou 1, 1 pour activer la communication entre les clients du vpn
  • <cert> : Nom du certificat serveur (eviter de changer cette valeur)
  • <masquerade> : prend pour valeur 0 ou 1, 1 pour activer l adresse IP du LCS sera utilise pour joindre l'interieur de l etablissement
  • <wins> : Adresse du serveur Wins pour les clients VPN, aucun desactive la fonctionnalite.
  • <dnsclient> : prend pour valeur 0 ou 1, 1 pour activer LCS comme DNS sur les clients VPN
