Dummy Plugin ou le Plugin LCS model

Un «Dummy Plugin», pourquoi faire ?

Pour faciliter le développement des plugins, une maquette de plugin respectant les spécifications structurelles du concept de «Plugin LCS» et proposant un model de script d'installation, désinstallation, mise à jour est mis à disposition des développeurs de plugins pour un savoir plus, consultez la documentation LCS (chapitre VII.) en suivant ce lien...
