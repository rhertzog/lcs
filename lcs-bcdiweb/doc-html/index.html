Module Lcs_Bcdiweb

Le module BcdiWeb permet d'installer facilement BcdiWeb version 2.04 sur le serveur Lcs. Une fois le module installé et le code d'activation de bcdiweb rentré, le mécanisme s'appuie sur une recopie de la base bcdi via le logiciel libre Rsync. La mise a jour se fait en automatique à une heure donnée, ou bien en manuel. La mise en place de cette recopie constitue de fait une sauvegarde automatique supplémentaire de la base BCDI.

Sa mise en place se fait en 3 étapes:

  • Installation du module et paramétrage de BcdiWeb
  • Installation du serveur Rsync sur le serveur BCDI pour la réplication de la base
  • Réplication de la base

Installation du module

Une fois le module BcdiWeb installé, un nouveau menu "BCDI WEB" apparait dans le menu d'administration de Lcs. La première étape consiste à cliquer dans Configuration pour saisir le code d'activation du BCDI WEB et l'adresse IP du serveur BCDI pour la synchronisation des données.

Installation du serveur RSYNC

Allez sur le serveur hébergeant la base BCDI. Celui-ci doit être sous Windows 2000/XP. Depuis ce poste, allez dans le menu administration de Lcs / BCDI WEB / Serveur Bcdi. Suivez alors les instructions afin d'installer et configurer le service Rsync sur le poste.

Réplication de la base

Une fois le service Rsync installé et configuré sur le poste hébergeant la base BCDI, allez dans le menu Administration Lcs / BCDI WEB / Mise a jour Base. Vous pouvez alors lancer une recopie immédiate de la base sur Lcs afin de tester tout de suite le fonctionnemment de bcdi web. Vous pouvez ensuite sélectionner une heure pour la recopie automatique de la base bcdi. Si la base BCDI est sur une machine allumée en permanence, une recopie la nuit conviendra parfaitement. Dans le cas contraire, il conviandra de choisir une heure où la machine est allumée et pas trop sollicitée.
