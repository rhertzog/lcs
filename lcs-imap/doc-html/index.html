Reconfiguration de courier-imap

Suite à l'installation du paquet lcs-imap, il est possible de reconfigurer courier-imap grace au script /usr/share/lcs/scripts/imap-reconfigure.sh.

Usage :

imap_reconfigure.sh 
    <LDAP_SERVER> <LDAP_BASE_DN> <LDAP_ADMIN_RDN> <LDAP_ADMIN_PW>

Définitions des paramètres :

  • <LDAP_SERVER> : Adresse IP du serveur LDAP
  • <LDAP_BASE_DN> : Base DN de l'annuaire
  • <LDAP_ADMIN_RDN> : RDN admin
  • <LDAP_ADMIN_PW> : Mot de passe admin annuaire LDAP
