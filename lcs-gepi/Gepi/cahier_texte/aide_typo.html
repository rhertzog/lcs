  • Gras, italique et souligné

    On indique simplement du texte en italique en le plaçant entre les symboles '' (c'est-à-dire, deux fois le symbole "apostrophe"). Ainsi, si vous tapez : « ''Voici du texte en italique'' », vous obtiendrez : « Voici du texte en italique ».

    On souligne du texte en le plaçant entre les symboles ''' (c'est-à-dire, trois fois le symbole "apostrophe"). Ainsi, si vous tapez : « '''Voici du texte en souligné''' », vous obtiendrez : « Voici du texte en souligné ».

    On indique du texte en gras en le plaçant entre les symboles __ (deux fois le caractère de soulignement).

  • Texte centré

    Il est possible de centrer du texte en le plaçant entre les symboles _c_.

  • Mettre des titres de différents niveaux

    On fait précéder les titres des symboles !, !!, ou bien !!!, selon la grosseur du titre souhaitée.

  • Mettre du texte en couleur

    On indique par exemple le texte à mettre en rouge en le plaçant entre les symboles _r_ . Ainsi, si vous tapez : « _r_Voici du rouge_r_ », vous obtiendrez : « Voici du rouge ».
    Autres couleurs reconnues :
    _b_ pour le bleu
    _v_ pour le vert
    _o_ pour l'orange
    _m_ pour le marron
    _j_ pour le jaune
    _p_ pour le pourpre
    _g_ pour le gris

  • Les liens hypertextes

    Les liens hypertextes sont directement reconnu par GEPI. Ainsi, si vous tapez : « http://www.google.fr. », vous obtiendrez « http://www.google.fr. ».
    De plus, on fabriquera facilement un lien hypertexte personnalisé avec le code suivant : « [Le moteur de recherche google|http://www.google.fr] » devient « Le moteur de recherche google ».
    De même il est possible d'insérer une image en tapant un code tel que celui-ci : [http://www.monsite.fr/image.jpg].

  • Retour à la ligne

    Un simple retour à la ligne dans le code ne provoque pas de retour à la ligne à l'écran.
    Pour obtenir un retour à la ligne :
    - ou bien il suffit de rajouter en fin de ligne les symboles %%% (trois fois le symbole "pourcentage"),
    - ou bien on laisse une ligne vide. Dans ce cas, le retour à la ligne sera accompagné d'un saut de ligne.

  • Insérer des balises html

    Il est possible d'insérer des balises html dans le texte, mais pour qu'elle soient interprétées correctement, il faut faire commencer chaque ligne par le symbole « | »
