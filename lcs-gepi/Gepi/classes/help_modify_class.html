GEPI - Gestion des classes : aide sur les matières

Sur cette page, vous définissez les matières enseignées dans la classe et les professeurs qui les enseignent.
1/ On commence par cocher les matières enseignées.
2/ Une fois que l'on a validé, on précise les professeurs qui enseignent la matière dans la classe. Il est possible d'affecter un nombre quelconque de profeseurs pour une matière. A chaque validation, un nouveau champ de saisie apparaît permettant l'affectation d'un nouveau professeur. ATTENTION : si aucun professeur n'est affecté, la matière n'est pas enregistrée !
3/ ensuite on peut (dé)sélectionner les élèves concernés par la discipline. Par défaut tous les élèves suivent la discipline, on peut cliquer sur le lien "Elèves inscrits" pour faire apparaitre une liste permettant de (dé)selectionner certains élèves.
  • Elèves inscrits

    Ce lien ouvre une fenêtre permettant de visualiser/modifier les options de chaque élève, pour la matière concernée.
    Le nombre entre parenthèses qui s'affiche à côté de la mention "Elèves inscrits" indique le nombre d'élèves inscrits dans la matières pour la première période.

  • Priorité d'affichage

    Permet de définir l'ordre d'affichage des matières dans le bulletin scolaire et dans les tableaux récapitulatifs des moyennes.
    Remarques :
    • Il est possible d'attribuer le même poids à plusieurs matières n'apparaissant pas sur un même bulletin. Par exemple, toutes les LV1 peuvent avoir le même poids, etc.
    • Si deux matières apparaissant sur un même bulletin ont la même priorité, GEPI affiche la première matière extraite de la base.


  • Coefficient

    Permet de calculer pour la moyenne générale de la classe et la moyenne générale de chaque élève. Ces valeurs sont affichées dans le bulletin scolaire.

    1/ Si tous les coefficients pour une classe sont égaux à zéro, il n'y a pas de calcul de la moyenne générale et, par conséquent, pas d'affichage d'une ligne supplémentaire sur le bulletin.

    2/ Si au moins un coefficient est non nul, une ligne supplémentaire est affichée sur le bulletin, indiquant les moyennes de l'élève et de la classe sur toutes les matières affectées d'un coefficient non nul.
